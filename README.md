# lunamachi

Rooms of tiny AI-generated beings that wander, pose, and greet each other —
with a fancy-Winamp lo-fi music player and a frequency visualizer. Each room is
its own URL (`/white`, `/moon`, …). No ads, no feeds. Just music and little
beings to watch. Live at [lunamachi.sprited.ai](https://lunamachi.sprited.ai).

A standalone static app (extracted from `sprite-dx`). Pure client side: Pixi.js
renders the room, the Web Audio API drives the visualizer, assets are served
from `public/`. No server/worker needed.

## Design

**No assets — every prop is procedurally generated.** The moon, stars, magic
circles, floors, glows: all drawn in code, no art files. The only file-based
content is the AI-generated characters and music. See [PRINCIPLES.md](./PRINCIPLES.md).

## Develop

```bash
npm install
npm run dev      # http://localhost:5180
npm run build    # → dist/  (static; deploy on Cloudflare Pages / any static host)
```

Press **Tab** in the room for a debug HUD (size, perspective, floor, shadow,
speed, pose/greet chance, population).

## Assets

The beings are characters generated by the `sprite-dx` pipeline; the player
serves lo-fi tracks. Committed assets live under `public/`:

- `public/beings/<id>/` — `sprites.webp` (pixel spritesheet) + `entity_pixel.json` (manifest)
- `public/beings.json` — the index the room loads
- `public/music/` — `*.mp3` + `playlist.json`

### Refreshing the cast

Drop full-size character exports (each a generation run dir with
`spritesheet_pixel.png` + `entity_pixel.json` [+ `reference_rgba.png`]) and:

```bash
python3 scripts/populate-beings.py --from '/path/to/runs/*' --count 120   # → public/_raw/ (gitignored)
python3 scripts/optimize-beings.py                                        # → public/beings/ + beings.json
```

`populate` rejects characters that bleed alpha to the cell edge (generation
errors); `optimize` re-encodes the sheets as WebP and crops the reference pose.
Requires Python + Pillow.

### Adding music

Drop an `.mp3` into `public/music/` and add an entry to `public/music/playlist.json`.
